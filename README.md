# Архив моей генеалогической информации

Этот репозиторий — эксперимент по долговременному хранению семейных/генеалогических материалов с использованием GitHub и GitHub Pages.  
Сейчас здесь размещён экспорт из Notion в простых форматах, чтобы данные можно было просматривать без специализированных программ и (в идеале) находить через поисковые системы.

## Зачем это нужно

Цель эксперимента — проверить, можно ли организовать хранилище, которое одновременно:

1. **Долговечное и надёжное** (контроль версий, история изменений, резервирование).
2. **Практически бесплатное** (или с минимальной стоимостью).
3. **Индексируется поисковиками** (чтобы информацию можно было найти обычным поиском).
4. **Использует простые открытые форматы** (Markdown/HTML/текст/изображения), не требующие узких приложений.

## Что находится внутри

- Экспорт/копия данных из **Notion**  
- Структурированные материалы (страницы, заметки, документы, изображения — в зависимости от экспорта)
- История изменений в Git (когда и что добавлялось/правилось)
